# InkEcho

A calm and writer-friendly blogging platform built with Django. Designed to be a peaceful space for writers to share their thoughts and for readers to enjoy quality content — without algorithmic pressure or distractions.

## About the Project

InkEcho allows writers to create rich blog posts and readers to discover and engage with meaningful content. The platform focuses on simplicity, good typography, and a warm reading experience.

This project is part of my journey as a Part III Computer Science student at Obafemi Awolowo University to build practical full-stack web applications.

## Current Features (Blogging System)

- User registration and authentication (Reader / Writer roles)
- Rich text editor using **CKEditor 5** (supports image upload, text alignment, captions, and media embedding)
- Create, edit, publish, and search blog posts
- Responsive and clean UI built with **Tailwind CSS**
- PDF generation for blog posts using WeasyPrint
- Modern, calm design with warm color palette

## Technologies Used

- **Backend**: Django (Python)
- **Rich Text Editor**: django-ckeditor-5
- **Frontend**: Tailwind CSS
- **Database**: SQLite (ready for PostgreSQL)
- **Tools**: Git, VS Code

## Project Status
